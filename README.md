# Python考研数据分析系统 python211

## 项目概述

本项目是一个基于Python语言的考研数据分析系统。它集成了院校评分、专业查询、分数线与报录比分析等功能，为考研学生提供全面的信息参考和决策支持。

## 技术栈

- 后端：Django
- 数据库：MySQL
- 前端：HTML + CSS + JavaScript

## 功能模块

### 用户端

- 院校评分
- 查专业
- 查分数线
- 查报录比
- 院校推荐
- 数据可视化
- 我的评分
- 我的收藏

### 管理端

- 院校管理
- 专业管理
- 分数线管理
- 报录比管理
- 用户管理
- 收藏管理
- 评分管理

## 系统角色

- 普通用户：查询和分析数据，进行个人评分和收藏
- 管理员：管理后端数据，维护系统运行

## 运行环境

- Python环境
- Django框架
- MySQL数据库
- 支持HTML、CSS、JavaScript的Web浏览器

## 部署步骤

1. 环境配置：安装Python、Django、MySQL
